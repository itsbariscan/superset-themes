# Inkwell — moodboard

**Concept:** Fountain-pen blue ink dried on bone-white paper. Long-form writing surface. The opposite of a blinking cursor — patience.

**Anchor references:** A thick journal, Pilot Iroshizuku ink, a Bauer Bodoni printed page, late-night essay drafts.

**Anchor colors:**
- Bone paper: `#f7f3eb` (warm white, slightly cooler than Vellum)
- Ink: `#1c2840` (deep blue-black — the foreground)
- Pen blue: `#2c4a8a` (primary — saturated but not garish)
- Annotation red: `#9c3a3a` (destructive — old margin notes)
- Quill grey: `#5b6a85` (mutedForeground — pencil annotation)
- Border: `#d4cdb8`
- Highlight: `#cda85c` (gold leaf — for emphasis)

**Mood:** Light theme, but distinctly different from Vellum. Vellum is sepia/warm-earth; Inkwell is cool-paper with a single saturated blue. Where Vellum reads as "manuscript," Inkwell reads as "essayist's draft." Distinct enough to ship both.

**What this avoids:** Excess saturation, neon, magenta. Stays in the literary register.

**Type:** light

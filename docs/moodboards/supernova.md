# Supernova — moodboard

**Concept:** Hubble photograph of a star going off. Deep-space black, nebula magentas, oxygen-emission teal, ionized-hydrogen pink. Astronomical photometry, not Photoshop "space."

**Anchor references:** Hubble's Pillars of Creation, the Crab Nebula in narrowband, Cassini's Saturn pole. Specifically the *narrowband* (Hubble palette: SII / Hα / OIII) renders, where each gas emission gets its own band.

**Anchor colors:**
- Deep space: `#040712` (the void)
- Foreground: `#dee5f5` (starlight white, cold cast)
- Nebula magenta: `#d24aff` (primary — ionized hydrogen)
- Oxygen teal: `#3fcfd8` (chart3 — OIII band)
- Hα pink: `#ff5db4` (chart1 — bright nebula gas)
- Sulfur orange: `#ffa040` (chart4 — SII band)
- Stellar yellow: `#fff0a8` (highlight — close star)
- Card / nebular shadow: `#0c1023`
- Border: `#1a2040`

**Mood:** Dark, dramatic, but anchored in real astrophotometry — every accent corresponds to an actual emission line. Different from Tokyo Night (city neon) and Twilight Run (atmospheric moment): Supernova is *astronomical*. Distinct because the chart palette is the Hubble narrowband filter set.

**Type:** dark

# Vellum — moodboard

**Concept:** A bound notebook on a desk by a window. Cream paper that has lived a little. Ink, not pixels. The sense of writing something to keep.

**Anchor references:** Field Notes paper stock, vellum manuscripts, Moleskine ivory pages, fountain-pen ink dried on cream. Think Edward Tufte's printing, McSweeney's typography, an old library reading room.

**Anchor colors (sampled):**
- Vellum cream: `#f5efdc` (warm paper, not white)
- Deep ink: `#1a1814` (almost black, but warm — never `#000`)
- Sepia red: `#8b3a1a` (single accent — pen ink that has aged)
- Olive scribble: `#6b5d3c` (secondary annotation)
- Parchment edge: `#c8b896` (border, dust jacket)
- Faded teal: `#4d6b6b` (annotation in a different pen)

**Mood checklist:**
- Light theme. Type-first. The page should look like it would print well.
- Single-accent discipline — sepia red for primary, everything else earth-tinted.
- No pure white. No pure black. No saturated chart colors.
- Charts should feel like ink swatches, not Slack reactions.

**What this theme avoids:** neon anything, blue/cyan dominance, magenta/pink.

**Type:** light

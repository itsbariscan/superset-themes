# Catppuccin Mocha attribution

**Original theme:** Catppuccin (Mocha flavor) by the Catppuccin Org
**Source:** https://github.com/catppuccin/catppuccin
**License:** MIT (see [LICENSE](./LICENSE))

This is a port of the Mocha flavor (the darkest of four — Latte, Frappé, Macchiato, Mocha) to the superset.sh theme schema. Hex values match the canonical Catppuccin palette spec at https://catppuccin.com/palette. The port uses Mauve (`#cba6f7`) as the primary action color, consistent with Catppuccin's default brand mapping. Other flavors may be added in a future release.

Catppuccin's contributing guidelines are respected — no logos or branded assets are altered or redistributed; only the documented hex values are used.

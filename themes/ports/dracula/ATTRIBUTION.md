# Dracula attribution

**Original theme:** Dracula by Zeno Rocha and contributors
**Source:** https://draculatheme.com / https://github.com/dracula
**License:** MIT (see [LICENSE](./LICENSE))

This is a port of the canonical Dracula palette to the superset.sh theme schema. The 8-color hero palette (Background, Foreground, Selection, Comment, Cyan, Green, Orange, Pink, Purple, Red, Yellow) is taken directly from the spec at https://draculatheme.com/contribute. Any deviations are constrained to mapping decisions where superset.sh has a token Dracula's spec does not (e.g., `chart4` mapped to Dracula Yellow).

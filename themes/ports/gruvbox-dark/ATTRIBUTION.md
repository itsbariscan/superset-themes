# Gruvbox Dark attribution

**Original theme:** Gruvbox by Pavel Pertsev
**Source:** https://github.com/morhetz/gruvbox
**License:** MIT (see [LICENSE](./LICENSE))

This is a port of Gruvbox Dark (the "medium" contrast variant) to the superset.sh theme schema. Hex values match the canonical Gruvbox palette spec. The Gruvbox primary action color is mapped to `bright_yellow` (`#fabd2f`) which is the canonical "selected" / "highlight" color in Pertsev's design.

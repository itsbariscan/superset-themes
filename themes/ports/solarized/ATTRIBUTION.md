# Solarized attribution

**Original theme:** Solarized by Ethan Schoonover
**Source:** https://ethanschoonover.com/solarized / https://github.com/altercation/solarized
**License:** MIT (see [LICENSE](./LICENSE))

This is a port of the Solarized 16-color palette (8 base monotones with engineered luminance pairs + 8 accents that retain perceptual identity across both bases) to the superset.sh theme schema. Both `solarized-dark.json` and `solarized-light.json` use identical accent hues — only the four base tones flip per Schoonover's design.
